HOW TO GENERATE THE REPORT
================================================================================

METHOD 1: Using Super Auditor (Recommended)
--------------------------------------------
1. Launch MAT: .\Start-MAT.ps1
2. Connect to tenant: Press [C]
3. Run Super Auditor: Press [4]
4. Confirm execution when prompted
5. Wait for all operations to complete
6. HTML report is automatically generated

The report will be saved as:
Downloads/MAT_Reports/<TenantName>/<Date>/MAT_Executive_Report_YYYYMMDD_HHMMSS.html

METHOD 2: Manual Execution (Advanced)
--------------------------------------
If you've already run individual audit modes:

1. Ensure you have these CSV files in the report directory:
   - Auditor_Report.csv
   - Protector_Inventory.csv
   - Licenses_Inventory.csv
   - Defensive_Stack.csv

2. Run in PowerShell:
   $reportPath = "C:\Path\To\Reports"
   $tenantName = "YourTenant"
   New-MATHtmlReport -ReportDirectory $reportPath -TenantName $tenantName
